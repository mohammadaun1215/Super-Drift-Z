# Super-Drift-Z

Super Drift Z is a basic San Andreas Multiplayer Stunt Gamemode released back in 2018 by $ilent[G]uy, perfect for hangouts.

## Commands

### General Commands
- `/kill`
- `/heal`
- `/armour`
- `/suggest`
- `/credits`
- `/build`
- `/rules`
- `/givecash`

### Vehicle Commands
- `/v`

### Teleport Command
- `/teles` (List of All Teleports)

### Deathmatch Commands
- `/dm`
- `/dm2`
